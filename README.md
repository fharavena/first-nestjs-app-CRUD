## Installation

``` MySQL
$ CREATE DATABASE sendmeapp;
$ CREATE USER 'nuevo'@'localhost' IDENTIFIED BY 'app';
$ GRANT ALL PRIVILEGES ON `borrame`.* TO 'nuevo'@'localhost';
```

```bash 1: back-end
$ cd api-nest-react
$ cd back
$ npm install
$ npm run start
```

```bash 2: front-end
$ cd api-nest-react
$ cd front
$ npm install
$ npm start
```
